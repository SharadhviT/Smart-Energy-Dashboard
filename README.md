# Smart Energy Awareness & Optimization Dashboard

## Description
This Streamlit app allows interactive input of household energy data in INR. Users can input multiple fields for each household and visualize:

- Daily Energy Usage (kWh)
- Daily Cost (₹)
- Households implementing energy-saving tips
- Average energy usage per occupant
- Percentage of households using LED bulbs

The app is fully interactive and suitable for engineering portfolio projects, demonstrating data visualization, analysis, and optimization skills.

## How to Run

1. Clone this repository:
   git clone <your_repo_url>
   
2. Navigate to the folder:
   cd smart_energy_dashboard
   
3. Install dependencies:
   pip install -r requirements.txt

4. Run the app:
   streamlit run smart_energy_dashboard.py

5. Input the household data and explore the charts and insights.
