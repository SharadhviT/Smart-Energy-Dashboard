# Smart Energy Awareness & Optimization Dashboard – 100 Households

## Description
Interactive Streamlit dashboard for 100 households to monitor energy usage (kWh) and cost (₹), analyze adoption of energy-saving tips, LED and AC usage, renewable energy adoption, and other household features.

## Features
- Line chart: Daily Energy per Household
- Bar chart: Cost per Household
- Pie charts: Tips Implemented, Renewable Adoption
- Bar chart: Energy per Occupant
- Bar chart: Average Energy by Household Type
- Boxplot: AC Usage vs Energy
- Top 5 energy-saving tips and insights summary

## How to Run
1. Clone the repository
   git clone <your_repo_url>
   cd smart_energy_dashboard
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   streamlit run smart_energy_dashboard.py
4. Explore interactive dashboard and charts.
